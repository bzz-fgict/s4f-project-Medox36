Pomodoro

Timer

A simple Pomodoro Timer made using Java Swing. For all who wish to use the pomodoro technique. The program can be customised to some extent by the Settings. Each Tab has the time a start/stop button and a reset button.

The Looks

The program is divided into three tabs, one for the pomodoro one for the short break and one for the long break.
At the top there is the possibility to change the settings.

The Program

When the Pomodoro timer is finished there will be a sound played. When nothing else is configured the short break will automatically start. And when the short break is finished, the pomodoro will start again.
Every three short breaks there is a long break instead.

When the program is closed, by default the program terminates, but the program can be customised to minimize itself to the system tray, if running on Windows. If minimizing to system tray is disabled again the GUI will show itself again without loosing progress.

The ability to list your tasks and manage them might be added in the near future.

The Settings

I'll go through the settings top to bottom.

First we have the two options to activate or disable the automatic start of short and long breaks ant pomodoros.

Next we have the option the change the duration of each of the three timers. And the option the change the interval between the short and long breaks. For example if the interval is set to 3 (default) every third break will be a long break.

Next we have the option to choose the sound of the alarm, when a timer is finished. Currently, there are two sounds available, a kitchen timer ringing and a bell. There might be added more sounds in the future.

Next we have the option th minimize the application to the system tray. This only works on Windows!

Last but not least whe have te option to save the settings to a specific folder and load settings.
Settings are automatically saver and loaded!

If the settings are altered from the default settings the program will automatically save the Settings in the same folder the program is located in.
