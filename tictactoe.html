Tic Tac Toe

A Spilme Tic Tac Toe game made using Java Swing.

The Looks

The game is kept simple. At the top there is a reset button, and it is displayed whose turn it is. The Game also works with touchscreens.

I've decided not to add a score count, but that might be added in the near future.
When a player wins, the three fields are marked green and a text is displayed which player wins.
